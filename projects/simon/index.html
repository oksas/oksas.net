Simon

A recreation of the classic game Simon for an assignment on Free Code Camp.

The design deviates significantly from that of Simon traditionally, but I knew from the outset that I wanted something that looked less like an actual Simon toy and more like a modern interface (though perhaps, deep down, this was really just an attempt to make Simon seem like more interesting of a game than it actually is).

Built with HTML, CSS (Sass), and vanilla JavaScript.
